üìãOverview

This powerful tool allows you to insert custom text (date, day, and month) into multiple PDF files simultaneously. The text is placed on white rectangles that intelligently cover any existing content at specified locations.

üí° Key Benefits:
  • Process multiple PDFs at once
  • Automatically covers old text with white backgrounds
  • Easy-to-use graphical interface
  • Preserves original PDFs

üì¶Installation Steps

Step 1: Create Project Folder

Create a new folder on your computer (e.g., C:\PDF_Text_Inserter)

Save these two files in this folder:

  • pdf_text_inserter.py - Main Python script
  • Run.bat - Batch file to run the tool

Step 2: First-Time Setup

  1. Double-click Run.bat
  2. Wait for automatic PyMuPDF library installation (first time only)
  3. Installation complete!

üöÄHow to Use

Step 1: Prepare Your PDFs

  • Place all PDF files you want to modify in one folder
  • Create an empty folder for output files

Step 2: Run the Tool

Double-click Run.bat

Step 3: Select Input Folder

Browse and select the folder containing your PDF files, then click "Select Folder"

Step 4: Select Output Folder

Browse and select where you want the modified PDFs saved

üí° TIP: Use a different folder than input to keep original files safe!

Step 5: Enter Text Information

Three dialog boxes will appear in sequence:

  1. Date Input: Enter the date (e.g., 11/11/2025)
  2. Day Input: Enter the day (e.g., 11)
  3. Month Input: Enter the month (e.g., November)

Step 6: Wait for Processing

The tool will process all PDFs. You'll see progress messages in the command window.

Step 7: Complete!

‚úÖ Check your output folder for the modified PDFs

Press any key to close the window

‚ùóTroubleshooting

Problem: "Python is not installed or not in PATH"
Solution:
  • Reinstall Python
  • Check "Add Python to PATH" during installation
Problem: "Failed to install PyMuPDF"
Solution:
  • Open Command Prompt as Administrator
  • Run: pip install PyMuPDF
Problem: No dialog boxes appear
Solution:
  • Check if antivirus is blocking the script
  • Run Run.bat as Administrator
Problem: Text appears in wrong location
Solution:
  • Open one PDF to check exact coordinates
  • Edit x and y values in pdf_text_inserter.py
Problem: Rectangle doesn't cover all text
Solution:
  • Increase rect_width or rect_height values
  • Edit values in pdf_text_inserter.py

üìùImportant Notes

  • Original PDFs are NOT modified - only output folder contains changes
  • All PDFs in input folder will be processed - no individual selection needed
  • Output files have the same names as input files
  • White rectangles cover existing content at specified locations
  • Page numbers start from 0 (Page 0 = first page)
